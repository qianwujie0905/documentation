## Start your Project

<AccordionGroup>
  <Accordion title="Prompt">
    Describe what you want to build in the prompt—be as specific as possible.
    <blockquote>
      Example: Create a dashboard with login, a line chart for monthly sales, and a pie chart for user segments.
    </blockquote>
  </Accordion>
  <Accordion title="Templates">
    Start from a template, then refine features and styling via prompts.
  </Accordion>
  <Accordion title="Remix an Existing Project">
    Remix from public projects (projects connected to Supabase are not supported yet).
  </Accordion>
  <Accordion title="Using a Sketch">
    Draw a sketch in Excalidraw (or similar), screenshot and paste—it becomes an editable code structure.
  </Accordion>
  <Accordion title="Cloning a Website or Application">
    Screenshot the target page and paste it. Superun reconstructs the structure for further editing.
  </Accordion>
</AccordionGroup>

## Edit your Project

<AccordionGroup>
  <Accordion title="Custom knowledge">
    Store product goals, design rules, and key context as custom knowledge to guide AI edits.
  </Accordion>
  <Accordion title="Select & Visual Edit">
    Use visual + selection edits to fine‑tune UI (content, colors, spacing, responsiveness, etc.).
  </Accordion>
  <Accordion title="Revert a Change">
    Roll back to any version via history; bookmark important edits.
  </Accordion>
  <Accordion title="Connect to GitHub">
    Connect GitHub to collaborate via repo or local IDE and keep in sync with Superun.
  </Accordion>
</AccordionGroup>

## Add Fullstack Capabilities

<AccordionGroup>
  <Accordion title="Connect a Backend（Supabase）">
    Create a project in Supabase → connect in Superun (project URL and API Key) → create tables/RLS/notifications, etc.
  </Accordion>
  <Accordion title="Authentication with Supabase">
    Use Superun’s AI form capabilities to build login/registration; configure Edge Functions and session management.
  </Accordion>
</AccordionGroup>

## Deploy & Publish

<AccordionGroup>
  <Accordion title="Set up SEO">
    Complete page metadata and social images following Superun recommendations.
  </Accordion>
  <Accordion title="Add a Custom Domain">
    Add a domain via Vercel/Netlify/Namecheap, etc.
  </Accordion>
  <Accordion title="Publish my Project">
    When ready, publish to get a shareable link; republish after further changes.
  </Accordion>
</AccordionGroup>
