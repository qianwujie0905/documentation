Kick Start allows you to generate complete, editable design drafts directly from plain text.  
Describe your design intent in natural language, and Prompt.to.design automatically creates a structured layout inside Figma.  
You can also upload reference images, PDFs, or documents to guide the generation process.

![Kick Start](/images/kick-start.png)

## Overview

Prompt.to.design interprets text descriptions as layout instructions. It analyzes your content and transforms it into a coherent, visually balanced Figma frame.  
The system automatically applies component-based hierarchy, spacing, and alignment to create ready-to-edit wireframes or high-fidelity mockups.

## Key Capabilities

- Generate new design drafts from text prompts  
- Understands both short and detailed instructions  
- Supports uploading external references for context  
- Automatically applies consistent alignment and spacing  

## When to Use Kick Start

Use Kick Start when you need to start a new design quickly without manual layout work. It is ideal for early-stage ideation, client concept visualization, and team collaboration.
