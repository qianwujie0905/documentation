---
title: Kick Start
description: Generate a complete layout instantly using a simple text prompt.
---
Kick Start helps you create a ready-to-edit Figma layout from just one short description.  
It's the fastest way to move from idea to visual design in seconds.

---

## 1. Click "Kick Start"
Open the plugin and click **Kick Start** to begin.

<img src="/images/kick-start/button.png" alt="Kick Start button" width="520" />

---

## 2. Describe what you want
Enter a simple prompt describing the layout you want.

Examples you can try:
- "Clean resume landing page"
- "Dashboard with charts and sidebar"
- "Mobile sign-up screen with hero image"

<img src="/images/kick-start/prompt.png" alt="Prompt input" width="520" />

---

## 3. Click Generate
The plugin creates a complete, editable layout using Auto Layout and Figma components.

<img src="/images/kick-start/sample.png" alt="Generated sample" width="520" />

---

## Tips for Best Results

- Keep prompts short and clear  
- Describe structure, not small visual details  
- Add style keywords if needed: "minimal", "modern", "bold typography"

---

## What You Can Do Next

After Kick Start generates your layout, refine it with:

- **Style Transfer** - match visual styles  
- **Region Design** - redesign specific areas 
- **Smart Fill** - fill empty spaces intelligently  

Kick Start gives you the foundation - the rest is easy.
