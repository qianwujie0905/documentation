Modify Elements provides precise control over existing components.  
Using short text instructions, designers can make changes such as editing text, adjusting colors, or modifying spacing.  
Prompt.to.design interprets each instruction and applies updates directly to the selected Figma elements.

![Modify Elements](/images/main.png)

## Overview

This capability simplifies micro-iterations during design reviews.  
It ensures all edits respect grid alignment, responsive behavior, and established design constraints, eliminating the need for manual resizing or correction.

## Key Capabilities

- Modify text, colors, and typography via command input  
- Maintain layout stability and component hierarchy  
- Enable efficient visual iteration in Figma  
- Integrate seamlessly with Kick Start and Region Design  

## When to Use Modify Elements

Use Modify Elements for detailed adjustments during the design refinement phase. It allows teams to update visuals quickly while preserving structural consistency.
