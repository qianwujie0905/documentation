# Style Transfer

Style Transfer enables you to create new pages that match the visual language of your existing designs.  
By selecting a Figma Frame as a reference, Prompt.to.design learns its color palette, typography, and spacing system, then applies these attributes to new pages automatically.

![Style Transfer](/images/style-transfer.png)

## Overview

This feature ensures brand and visual consistency across large projects.  
Whether building additional product pages, dashboards, or mobile layouts, Style Transfer preserves your established design identity.

## Key Capabilities

- Learns design tokens such as color, typography, and layout  
- Generates new pages with consistent visual tone  
- Works across web and mobile frameworks  
- Reduces repetitive manual styling tasks  

## When to Use Style Transfer

Use Style Transfer when expanding a product's interface or maintaining design system consistency. It is especially effective for teams that manage multiple pages within the same visual identity.
